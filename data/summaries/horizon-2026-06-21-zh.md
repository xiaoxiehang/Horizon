## [SMPTE 免费开放其全部媒体技术标准库](https://www.smpte.org/blog/smpte-makes-its-standards-freely-accessible-openingstandards-library-to-the-global-media-technology-community) ⭐️ 8.0/10

电影电视工程师协会（SMPTE）已将其包含 800 多项媒体技术标准的完整标准库向公众免费开放。这一举措是其更广泛的现代化工作的一部分，包括采用基于 GitHub 的工作流和结构化的 HTML 创作方式。 取消 SMPTE 标准的财务壁垒，显著降低了从事媒体制作和广播的开发者、研究人员及小型公司的准入门槛。这种开放获取模式有望加速全球媒体技术生态系统的创新和互操作性，类似于 IETF 等开放框架所取得的成功。 除了免费访问外，SMPTE 还在通过转向基于 GitHub 的版本控制、自动化问题跟踪以及用于文档验证和发布的集成发布管道来现代化其发布流程。此前，专业人士必须购买单独的标准 PDF（例如 430.10 标准）才能实现特定的技术集成。

hackernews · zdw · 6月20日 17:01 · [社区讨论](https://news.ycombinator.com/item?id=48610827)

**背景**: SMPTE 是一家国际公认的标准组织，为娱乐和媒体行业提供至关重要的技术框架。其庞大的目录涵盖了从时间码、镜头卡口到影院声音和数字视频格式等各个方面，确保了高度复杂的全球供应链中的互操作性。从历史上看，获取这些关键文档需要付费会员资格或单独购买文档。

## [UHF X11 将传统 X11 协议引入 Apple Vision Pro](https://www.lispm.net/apps/uhf-x11/) ⭐️ 6.0/10

一位开发者成功将拥有 40 年历史的 X11 显示协议移植到了苹果的 visionOS，为 Apple Vision Pro 开发了 UHF X11 应用。这使得用户能够在苹果的空间计算环境中运行和查看经典的 X11 应用程序。 虽然这主要是一个技术噱头和复古计算爱好者的怀旧之举，但它展示了 visionOS 在处理传统显示协议方面的灵活性。这也突显了社区对于在现代空间计算硬件上运行传统 Linux 和 Unix 桌面环境的持续兴趣。 该应用专为复古计算、实验和教育而设计，但它不加密 X11 流量，也不包含内置的安全隧道。用户可以通过 X11 使用 GLX 渲染来支持 OpenGL 客户端，不过其兼容性各不相同，就像 21 世纪初的情况一样。

hackernews · zdw · 6月20日 17:04 · [社区讨论](https://news.ycombinator.com/item?id=48610853)

**背景**: X11（也称为 X 窗口系统）是位图显示的基础网络透明窗口系统，几十年来广泛应用于类 Unix 操作系统。Apple Vision Pro 运行的是 visionOS，这是一种空间操作系统，通常依赖现代原生框架而非传统的 Unix 显示服务器，因此这次移植成为了连接新旧计算范式的独特桥梁。

## [经典 DOS 游戏《F-15 Strike Eagle II》逆向工程项目招募测试者](https://neuviemeporte.github.io/f15-se2/2026/06/20/needyou.html) ⭐️ 6.0/10

一位开发者正在将经典 DOS 游戏《F-15 Strike Eagle II》从汇编语言逆向工程为 C 代码，目前正在招募测试者以在移植到现代操作系统之前查找 DOS 版本中的缺陷。 该项目凸显了将遗留汇编代码转换为现代可移植 C 代码所涉及的技术复杂性和保护工作，确保经典游戏能够在当前系统上原生运行，而无需依赖模拟器。 逆向工程过程首先将游戏转换为汇编代码，然后在 DOS 上将其转换为二进制等效的编译 C 代码，需要测试者拥有 451.03 版本的游戏以及 DOSBox 或真实的 DOS 硬件。

hackernews · LowLevelMahn · 6月20日 15:10 · [社区讨论](https://news.ycombinator.com/item?id=48609766)

**背景**: 逆向工程编译后的 DOS 游戏涉及将其机器码或汇编代码解构为 C 等高级语言，以理解其逻辑并最终为现代平台重新编译。通常使用 DOSBox 等模拟器在现代系统上运行这些老游戏，但原生移植能提供更好的性能和集成度。

**社区讨论**: 社区成员讨论了逆向工程过程的技术步骤，并质疑在 DOSBox 等模拟器已经能良好运行游戏的情况下进行反编译的必要性。其他人则表达了怀旧之情，并探讨了利用 AI 帮助理解缺乏符号名称的反编译代码结构的潜力。
