科学家在链霉菌中发现了一个罕见的抗生素巨型基因簇，该基因簇能产生四种针对细菌生物素代谢的不同抗生素家族。这一发现将抗生素发现的范式从分离单一化合物转变为理解自然进化的联合疗法。 这一突破通过提供联合疗法来补充抗生素库，为应对日益严重的抗生素耐药性超级细菌全球威胁提供了充满希望的新策略。它突显了基因组挖掘在发现能够克服现有耐药机制的复杂多化合物天然产物方面的巨大潜力。 该巨型基因簇专门产生四种共同靶向细菌生物素代谢的化合物，代表了一种自然进化的联合疗法。这种架构支持药物发现战略向靶向整个生物合成基因簇而非单一基因靶点转变。
