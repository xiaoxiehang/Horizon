一位开发者正在将经典 DOS 游戏《F-15 Strike Eagle II》从汇编语言逆向工程为 C 代码，目前正在招募测试者以在移植到现代操作系统之前查找 DOS 版本中的缺陷。 该项目凸显了将遗留汇编代码转换为现代可移植 C 代码所涉及的技术复杂性和保护工作，确保经典游戏能够在当前系统上原生运行，而无需依赖模拟器。 逆向工程过程首先将游戏转换为汇编代码，然后在 DOS 上将其转换为二进制等效的编译 C 代码，需要测试者拥有 451.03 版本的游戏以及 DOSBox 或真实的 DOS 硬件。
